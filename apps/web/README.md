# Web

Next.js application with Tailwind CSS and shared UI components.

## Development

```bash
pnpm dev --filter web
```

Runs on [http://localhost:3000](http://localhost:3000).

## Environment Variables

Copy `.env.example` to `.env.local`:

```bash
cp .env.example .env.local
```

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- TypeScript
